# Internship Capacity Planning

## Overview

This project analyzes internship capacity by combining internship start and end dates.

The goal is to support future internship planning by identifying department occupancy over time.

## Features

- Weekly capacity analysis
- Department occupancy
- Internship timeline
- Planning support

## Business Value

Provides data-driven insights for future internship placement decisions and workload balancing.

## Tools

- Microsoft Excel
- Pivot Tables
- Conditional Formatting

> Note: Sample data is fictional and created for demonstration purposes.
